Notes:

Every html5 document should start with <'!DOCTYPE html> and be written inside <'html>...<'/html>.
Each html document should have a head (<'head>...<'/head>) and a body (<'body>...<'/body>).
<'h1> is largest, <'h6> is smallest header.
<'br> is break line (no end element).
<'hr> is horizontal rule (horizontal line).
<'p>...<'/p> is for paragraphs.
Each element can have their id. Example: <'p id="...">...<'/p>
<'!-- ... --> is for comments.
<'a href="link here"..>...<'/a> is for links, you can also email with mailto.
<'img src="..." height=... width=... (if only height is defined width will scale proportionately) alt=... title=...> is for images
<'audio controls autoplay muted loop src="..."> or <'audio controls><'source src="sounds/jingle.mp3">This browser does not support this audio format.<'/audio> is for audio (controls autoplay muted and loop are booleans)
Video works very similar to audio.
<'b>...<'/b> = bold text, <'i>...<'/i> = italic text, <'big>...<'/big> = big text, <'small>...<'/small> = small text, <'sub>...<'/sub> = subscript text, <'sup>...<'/sup> = superscript text, <'ins>...<'/ins> = inserted text, <'del>...<'/del> = deleted text and <'mark>...<'/mark> = marked text.
<'ul>...<'/ul> is for unordered lists, <'ol type=""(1,A,a...)>...<'/ol> is for ordered lists, <'li>list item here<'/li> is for list items (inside of an ordered/unordered list). You can make lists inside lists (nested lists).
<'dl>...<'/dl> is for description lists, <'dt>...<'/dt> is definition term and <'dd>...<'/dd> is definition description.
<'table bgcolor="">...<'/table> is for tables. Inside of tables we have <'tr bgcolor="" align="">...<'/tr> - table rows and inside of table rows we have <'th>...<'/th> - table headers and <'td>...<'/td> - table data.
CSS style attribute is used for coloring, for example: <'b style="color="yellow"><'/b>. Aside from color names you can also use hex (#number) and rgb (rgb(red, green, blue)) values.
span adds markup to text or a portion of a document. <'span>...<'/span>

div defines a division of a document <'div>...<'/div>

meta element represents various kinds of metadata that cannot be expressed using the title, base, link ,style and script elements. <'meta> is placed in the head part of document
iframe = embed content from another source into an HTML document. Example: <'iframe src="https://www.google.com">
<'button onclick="..." (when button is clicked) disabled (for disabling a button), style="...">...<'/button> is for buttons. You can put buttons inside <'a> (link) tags.
